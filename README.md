Unshorten
====

Get original URLs from shortened ones.

Install
----

    $ gem install unshorten

Usage
----

Using shell:

    $ unshorten bit.ly/vkmMJO
    https://github.com/quark-zju/unshorten

Using Ruby:

    require 'unshorten'
    url = Unshorten['bit.ly/vkmMJO'] # "https://github.com/quark-zju/unshorten"
